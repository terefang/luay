## Operating System Facilities

```lua
local os = require('os');
```

Refer: http://www.lua.org/manual/5.2/manual.html#6.9

*   `os.clock ()`
*   `os.date ([format [, time]])`
*   `os.difftime (t2, t1)`
*   `os.execute ([command])`
*   `os.exit ([code [, close])`
*   `os.getenv (varname)`
*   `os.remove (filename)`
*   `os.rename (oldname, newname)`
*   `os.setlocale (locale [, category])`
*   `os.time ([table])`
*   `os.tmpname ()`
