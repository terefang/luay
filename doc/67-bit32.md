## Bitwise Operations

```
local bit32 = require('bit32');
```

Refer: http://www.lua.org/manual/5.2/manual.html#6.7

*   `bit32.arshift (x, disp)`
*   `bit32.band (···)`
*   `bit32.bnot (x)`
*   `bit32.bor (···)`
*   `bit32.btest (···)`
*   `bit32.bxor (···)`
*   `bit32.extract (n, field [, width])`
*   `bit32.replace (n, v, field [, width])`
*   `bit32.lrotate (x, disp)`
*   `bit32.lshift (x, disp)`
*   `bit32.rrotate (x, disp)`
*   `bit32.rshift (x, disp)`
